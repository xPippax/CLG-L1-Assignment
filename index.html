HTML

  • Hyper Text Markup Language
  • made up of Tags and Elements
  • Can have Attibutes
  • Forms the bones or structure of a website
  • Tells the brower how to display content
  • Adds written content, images, tables and links, formats text and more

CSS

  • Cascading Style Sheet
  • Makes a website look pretty
  • Uses properties, selectors
  • Add/Change colours, fonts, fontsize,
  • Can be added within the index.html file or linked to an external style sheet
