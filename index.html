HTML

  • Hyper Text Markup Language
  • Made up of Tags and Elements
  • Can have Attibutes
  • Forms the bones or structure of a website
  • Tells the browser how to display content
  • Adds written content, images, tables and links, formats text and more

CSS

  • Cascading Style Sheet
  • Makes a website look great
  • Properties and Selectors
  • Add/Change colours, fonts, fontsize and more
  • Can be added inside the body of the index.html file or in a linked external style sheet
